### Description

Create a new plugin.

### Parameters

{:.table.table-striped.event-table}
| Option | Description | Required | Default |
| - | - | - | - |
| `--name` | the new plugin name | true | na|

### Examples

{:.table.table-striped.event-table}
| OS | Command |
| - | - |
| macOS / Linux  | `fulcrum create-plugin --name 'MyPlugin'`  |
| Windows | `.\fulcrum.cmd create-plugin --name "MyPlugin"` |
