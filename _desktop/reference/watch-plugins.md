### Description

Watch and recompile all plugins.

### Parameters

{: .desktop}
| Option | Description | Required | Default |
| - | - | - | - |
| `--name` | plugin name to watch | false | na|

### Examples

{: .desktop}
| OS | Command |
| - | - |
| macOS / Linux  | `fulcrum build-plugins`  |
| Windows | `.\fulcrum.cmd build-plugins` |
